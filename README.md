# vercel-ai-sdk-playground

Vercel AI SDK を使ってみる。

## Getting Started

1. `.env`の作成

   ```.env
   OPENAI_API_KEY="your_openai_api_key"
   ```

2. パッケージのインストールと localhost の立ち上げ

   ```bash
   npm install
   npm run dev
   ```

3. localhost にアクセス。

   [http://localhost:3000](http://localhost:3000)にアクセスすると一覧で機能を触れる。
